INFINITE MARIO balancing test

Below are links that lead to batches of Infinite Mario featuring different boredom/stress levels (game balancing).

Each batch has 3 games. You can jump to the next game of any batch by clicking the green CONCLUDE button at the bottom of the game page.

  • Batch 1 - boring: games A1 (extremely boring), B2 (a bit less boring than A1) and C3 (a bit less boring than B2). Subjects should perceive those games as boring.
  • Batch 2 - "ideal": games K11 (hopefully not boring nor stressful), L12 (a bit more challenging than K11) and M13 (a bit more challenging than L12). Subjects should perceive those games as "enjoyable". The games ideally should not be perceived as boring nor stressful.
  • Batch 3 - stressful: games T20 (extremely stressful), S19 (a bit less stressful than T20) and R18 (a bit less stressful than S19). Subjects should perceive those games as stressful.
  • Batch 4 - all games from boring to stressful: games A1 (extremely boring) to T20 (extremely stressful). This batch gives a better contextualized way of measuring the increase of boredom/stress throughout the games.

P.S. While playing any game, on top of the screen, the game HUD will show the "World name", e.g. "A1". This is a boredom to stress scale that goes as A1, B2, C3, D4, ..., T20. The higher the number, the more stressful the game should feel.
